## BTrace 용어
_Probe Point_
  일련의 추적 구문이 실행되는 "location" 또는 "event". Probe Point 는 우리가 추적하길 원하는 구문들이 있는 "place" 또는 "event" 에 관심이 있다.
_Trace Actions or Actions_
  실행되는 추적 구문들은 언제든지 중단될 수 있다.(?)
